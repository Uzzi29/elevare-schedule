# Elevaré — 12-Month Master Schedule

A personal webapp to track the full Elevaré 12-month plan across Brand, Purpose, and YouTube work days.

## Features
- **Tick ✓ / X ✕** — mark tasks complete or missed, saves automatically
- **Persistent across sessions** — your progress is saved in your browser's localStorage
- **Dashboard** — view all logged tasks at a glance
- **Phase navigation** — browse all 9 phases (M0 → M8)

## Live Site
> https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/

## How to deploy (GitHub Pages)

1. Create a new **public** GitHub repository
2. Upload both `index.html` and `.nojekyll` to the root
3. Go to **Settings → Pages**
4. Set source to: `Deploy from a branch` → `main` → `/ (root)`
5. Click **Save** — your site will be live in ~1 minute

## Notes
- Progress (ticks & X's) is saved per browser/device via localStorage
- Clearing browser data will reset progress
- Works on mobile and desktop
